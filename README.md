# Soup-Dino-Game
<br/>made for Fallout! [organised by Hackclub, 1st July - 7th July 2026]
<br/>  created by onlyth3best, blowupthenoobs, & trpalt… AKA, team **Soup in a Bag**.

<br/>Ever wanted to play the offline dinasour game in real life? Well, now's your chance!
<br/>Our magnificent contraption works with vibration sensors/accelerometers & XIAO RP2040s in order to detect movement and coordinate between microcontrollers.
